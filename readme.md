# JS Snack Functions

Questa repository contiene una serie di piccoli esercizi (snack) per fare pratica con le funzioni in JavaScript.

## Elenco Esercizi

- **01_saluti**: Crea una funzione che riceve un nome e restituisce una stringa di saluto (es. "Ciao Mario").

- **02_iniziali**: Crea una funzione che riceve un array di nomi e restituisce un nuovo array contenente solo l'iniziale di ogni nome.

- **03_filtra_iniziali**: Crea una funzione che riceve un array di parole e una lettera, e restituisce un nuovo array con solo le parole che iniziano con quella lettera.

- **04_conta_vocali**: Crea una funzione che riceve una stringa e restituisce il numero totale di vocali presenti.

- **05_bonus**: Crea una funzione che riceve un nome e restituisce un saluto che cambia in base all'orario ("Buongiorno", "Buon pomeriggio" o "Buonasera").
